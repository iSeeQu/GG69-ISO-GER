
# GG69 ISO
### Basically 65% US Layout Keyboard with a lubed ISO Enter key, which ANSI users miss. Plus and additional column for German umlauts: Ä, Ö, Ü

## Hardware-Specs
* MCU: STM32F072C8T6
* Layout: 65% ISO
* Matrix: 5 Rows x 16 Columns
