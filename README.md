
# GG69 ISO
## ANSI users are missing a perfect lubed ISO enter key...
### Basically a 60% US layout keyboard with ISO enter key and additional column for german Umlauts: Ä, Ö, Ü

## Hardware-Specs
* MCU: STM32F072C8T6
* Layout: 65% ISO
* Matrix: 5 Rows x 16 Columns
