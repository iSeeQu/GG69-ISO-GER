# GG69 ISO

## GG69-ISO-DEUS
### US layout with additional column for german Ä, Ö, Ü and ISO Enter Key

![DEUS LAYOUT](images/gg69_deus_keymaping.png)

### Hardware-Specs
* MCU:    STM32F072C8T6
* Socket: Kailh Hotswap
* Layout: 65% ISO
* Matrix: 5 Rows x 16 Columns
### Nonflex, case mounted PCB with Clip in Stabs
* Steped Caps Lock Key
* custom 4.25u Spacebar Key
* custom 2.50u Left Control Key
