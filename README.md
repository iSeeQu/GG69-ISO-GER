
# GG69 ISO
### Basically a 60% US keymaping with additional column for german Ä, Ö, Ü and ISO enter key

## Hardware-Specs
* MCU: STM32F072C8T6
* Layout: 65% ISO
* Matrix: 5 Rows x 16 Columns
